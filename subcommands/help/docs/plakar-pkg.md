PLAKAR-PKG(1) - General Commands Manual

# NAME

**plakar-pkg** - List installed Plakar plugins

# SYNOPSIS

**plakar&nbsp;pkg**

# DESCRIPTION

The
**plakar pkg**
lists the currently installed plugins.

# FILES

*~/.cache/plakar/plugins/*

> Plugin cache directory.
> Respects
> `XDG_CACHE_HOME`
> if set.

*~/.local/share/plakar/plugins*

> Plugin directory.
> Respects
> `XDG_DATA_HOME`
> if set.
